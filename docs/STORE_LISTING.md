# check Store Listing Draft

## 한국어

### 앱 이름

check

### 짧은 설명 (80자 이내)

매일 한 번 안전 체크인을 남기고, 놓치면 긴급 연락처에 빠르게 연락할 수 있는 앱

### 자세한 설명

check는 하루 한 번 간단히 생사 확인 체크인을 남길 수 있도록 돕는 가벼운 개인용 앱입니다.

체크인을 남기면 다음 마감 시간을 계산하고, 설정한 주기에 따라 다음 체크인까지 남은 시간을 보여줍니다. 체크인을 놓치면 앱 안에서 미체크인 경고를 보여 주고, 미리 저장해 둔 긴급 연락처에 문자 또는 이메일 작성 화면을 빠르게 열 수 있습니다.

주요 기능:
- 하루 체크인 버튼
- 체크인 이력 저장
- 긴급 연락처 추가 / 수정 / 삭제
- 문자 / 이메일 빠른 열기
- 12시간 / 24시간 / 48시간 체크인 주기 설정
- 알림 켜기 / 끄기
- 한국어 / 영어 전환
- 데이터 JSON 내보내기 및 초기화

이 앱은 서버 계정이나 백엔드 없이 동작하며, 사용자가 입력한 데이터는 기기 로컬 저장소에만 보관됩니다.

### 앱 카테고리 권장

건강/운동 또는 라이프스타일

### 개인정보처리방침 URL

실제 배포 전 공개 가능한 URL에 `docs/PRIVACY_POLICY.md` 내용을 게시한 뒤 해당 URL을 입력하세요.

## English

### App name

check

### Short description

Daily safety check-ins with quick emergency contact actions if a check-in is missed

### Full description

check is a lightweight personal app that helps users record a quick daily life-check confirmation.

After each check-in, the app calculates the next deadline and shows the remaining time based on the selected interval. If a check-in is missed, the app shows an in-app warning and lets the user quickly open the SMS or email composer for a saved emergency contact.

Key features:
- Daily check-in button
- Check-in history
- Add / edit / remove emergency contacts
- Quick SMS / email actions
- 12h / 24h / 48h interval settings
- Notifications on / off
- Korean / English language switch
- Export and reset data as JSON

The app works without a backend or account system, and user-entered data is stored only on the local device.
