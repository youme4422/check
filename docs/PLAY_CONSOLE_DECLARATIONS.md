# check Play Console Declarations

## 한국어

Play Console 입력 시 아래 기준으로 답변을 맞추는 것이 현재 앱 구조에 부합합니다.

- 광고 포함 여부: 아니요
- 계정 생성 필요 여부: 아니요
- 앱 액세스 제한: 없음
- 수집 데이터: 기기 로컬 저장만, 서버 전송 없음
- 데이터 보안 양식:
  - 수집됨: 사용자가 직접 입력한 연락처 정보, 앱 활동(체크인 기록)
  - 전송됨: 아니요
  - 삭제 가능: 예, 앱 내 초기화 기능 제공
- 알림: 로컬 알림만 사용
- 위치 정보: 사용 안 함
- 연락처 읽기 권한: 사용 안 함
- SMS 자동 전송: 사용 안 함
- 건강 앱으로 오해될 수 있는 의료 주장: 사용 금지

개인 개발자 계정이라면 Google Play의 최신 테스트 요구사항(예: 사전 테스트 트랙 운영)을 반드시 확인하세요.
