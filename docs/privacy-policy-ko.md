# 개인정보처리방침 (check)

## 1. 개요

"check"(이하 "앱")는 사용자의 일일 체크인 기록을 관리하는 안전 확인 앱입니다.

본 앱은 사용자의 개인정보를 외부 서버로 수집하거나 저장하지 않습니다.

## 2. 수집 및 저장 정보

본 앱은 다음 정보를 사용자 기기 내부에만 저장합니다:

- 체크인 기록 (날짜 및 시간)
- 사용자가 직접 입력한 긴급 연락처 정보
  - 이름
  - 전화번호
  - 이메일 주소

해당 정보는 외부 서버로 전송되지 않습니다.

## 3. 정보 처리 방식

- 모든 데이터는 사용자의 기기 내부 저장소(로컬 스토리지)에 저장됩니다.
- 별도의 서버, 클라우드 데이터베이스를 사용하지 않습니다.
- 개발자는 사용자의 데이터를 수집하거나 열람할 수 없습니다.

## 4. 제3자 제공

본 앱은 개인정보를 제3자에게 제공하지 않습니다.

단, 사용자가 직접 버튼을 눌러 문자 또는 이메일을 발송하는 경우, 해당 내용은 사용자가 선택한 SMS 또는 이메일 앱을 통해 전송됩니다.

앱은 자동으로 메시지를 전송하지 않습니다.

## 5. 권한 사용

본 앱은 다음 권한을 사용할 수 있습니다:

- 알림(Notification) 권한
  - 체크인 알림 제공 목적

본 앱은 다음과 같은 민감 권한을 사용하지 않습니다:

- 연락처 읽기
- 위치 정보
- 통화 기록
- 문자 메시지 읽기

## 6. 데이터 보관 및 삭제

- 모든 데이터는 사용자 기기에만 저장됩니다.
- 앱 삭제 시 관련 데이터도 함께 삭제됩니다.
- 앱 내 "데이터 초기화" 기능을 통해 언제든지 삭제할 수 있습니다.

## 7. 아동의 개인정보

본 앱은 14세 미만 아동을 대상으로 하지 않으며, 고의로 아동의 개인정보를 수집하지 않습니다.
