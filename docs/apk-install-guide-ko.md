# 시연 설치 안내

아래 APK 파일을 다운로드하거나 이 폴더에 있는 파일을 사용하면 Android 기기에서 바로 설치해 시연할 수 있습니다.

## 설치 방법

1. `taeb-demo.apk` 파일을 Android 기기에 전달합니다.
2. 파일을 열어 설치를 진행합니다.
3. 처음 설치할 때는 `알 수 없는 앱 설치 허용` 안내가 나올 수 있습니다.
4. 설치 후 `TaeB` 앱을 실행합니다.

## 다운로드 링크

## 참고

- 기존 `AAB` 파일은 Google Play 업로드용입니다.
- 일반 시연/설치는 `APK` 파일을 사용하는 것이 맞습니다.
