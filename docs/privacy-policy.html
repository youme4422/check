개인정보처리방침 (TaeB)

최종 업데이트: 2026-03-06

TaeB는 체크인 기록과 긴급 연락처를 기기 내부에 저장해 체크인 상태 확인과 수동 연락(SMS/이메일 앱 열기)을 돕는 앱입니다.

1. 수집 및 저장 정보

  • 체크인 시각/이력
  • 사용자가 직접 입력한 긴급 연락처 정보(이름, 전화번호, 이메일)
  • 앱 설정(체크인 주기, 알림 설정, 언어)

모든 정보는 기본적으로 사용자 기기 로컬 저장소에 저장됩니다.

2. 이용 목적

  • 체크인 상태 및 이력 표시
  • 사용자가 선택한 긴급 연락처로 빠른 연락 실행
  • 로컬 알림 예약 및 표시
  • 앱 설정 유지

3. 제3자 제공/외부 전송

  • 앱은 사용자 데이터를 개발자 서버로 자동 전송하지 않습니다.
  • 사용자가 직접 sms: 또는 mailto: 기능을 실행하면, 해당 데이터가 기기의 SMS/이메일 앱으로 전달될 수 있습니다.
  • 메신저 자동 발송(LINE/Telegram 등) 기능은 현재 앱에서 사용하지 않습니다.

4. 권한

  • POST_NOTIFICATIONS: 체크인 리마인더 알림 제공

연락처 읽기, 위치, 통화 기록, 문자 읽기/자동 발송 권한은 사용하지 않습니다.

5. 보관 및 삭제

데이터는 기기 로컬에 보관되며, 앱 초기화 기능으로 삭제할 수 있습니다.
