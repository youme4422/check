# App Showcase Site

`app-showcase`는 `check` 앱 소개용 정적 웹사이트입니다.

## 실행

1. `app-showcase/index.html`을 브라우저에서 열기
2. 또는 Live Server로 `app-showcase/` 실행

## 수정 포인트

- 메인 문구: `index.html`의 hero 섹션
- 기능 목록: `index.html`의 `#features`
- 스크린샷: `index.html`의 `#screens` 이미지 경로
- 디자인: `styles.css` 상단 `:root` 색상 변수
