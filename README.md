# check

## 한국어

`check`는 사용자가 매일 안전 확인 체크인을 남기는 Expo Go용 앱입니다.

### 실행 방법

1. 의존성이 없다면 설치합니다.

```bash
npm install
```

2. Expo 개발 서버를 실행합니다.

```bash
npx expo start
```

3. 터미널에 표시되는 QR 코드를 Expo Go 앱으로 스캔합니다.

### 주요 기능

- 매일 체크인 버튼과 남은 시간 표시
- 체크인 이력 보기 및 초기화
- 긴급 연락처 저장 / 수정 / 삭제
- 문자 / 이메일 작성 화면 바로 열기
- 체크인 주기, 알림, 언어 설정
- 데이터 JSON 내보내기

## English

`check` is an Expo Go app for daily life check-ins so a user can confirm they are OK each day.

### Run

1. Install dependencies if needed.

```bash
npm install
```

2. Start the Expo development server.

```bash
npx expo start
```

3. Scan the QR code shown in the terminal with Expo Go.

### Features

- Daily check-in button with countdown
- Check-in history list and reset
- Add / edit / remove emergency contacts
- Quick SMS / email composer actions
- Check-in interval, notifications, and language settings
- Export stored data as JSON
