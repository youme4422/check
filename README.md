# check

## Overview

TaeB is an Expo React Native safety check-in app.

## Run

1. Install dependencies.

```bash
npm install
```

2. Start Expo.

```bash
npx expo start
```

3. Scan the QR code with Expo Go.

## Main Features

- Daily check-in button with countdown
- Check-in history list and reset
- Messenger linking and emergency alert channels (LINE/Telegram/Email)
- Check-in interval, notifications, language, and theme settings
